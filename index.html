Resources

W3Schools
W3 Schools is an amazing resource to learn new and remember HTML, CSS, and JavaScript rules. I was first introduced to the website in CNIT 131H. We relied on the website to learn basic HTML. I rely on it often when I need to remember proper HTML syntax or CSS syntax. I would highly recommend reading through the first few HTML and CSS modules, because it will give you a proper, and brief introduction.
MDN Web Docs
MDN Web Docs, is another great resource. It goes much more in depth about HTML, CSS, and JavaScript. I also relied on the website to learn the basics of networking and how websites function. It is a great resource for people who want to learn about an aspect of web development and be provided with additional resources and topics that are relevant to the current topic.
